Ctrl + N | Новый файл

Ctrl + P | Быстрый переход к файлу

Ctrl + G | Перейти к строке

Ctrl + L | Выделить текущую строку

Ctrl + U | Убрать последний добавленный курсор

Ctrl + ` | Терминал

Alt + Shift + ↓ | Добавить курсор ниже
